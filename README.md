# Customer Behavior Analysis

## Overview
This project demonstrates an end-to-end Data Analytics workflow using Python, PostgreSQL, SQL, and Power BI. The project focuses on customer purchasing behavior analysis, data cleaning, exploratory data analysis (EDA), SQL querying, KPI analysis, and dashboard visualization.

---

## Tools & Technologies
- Python
- Pandas
- PostgreSQL
- SQL
- Power BI
- Jupyter Notebook

---

## Project Workflow
### 1. Data Loading
- Imported dataset into Python using Pandas

### 2. Exploratory Data Analysis (EDA)
- Analyzed customer behavior patterns
- Visualized trends and distributions

### 3. Data Cleaning
- Handled missing values
- Removed duplicates
- Corrected data types

### 4. SQL Analysis
Performed:
- Joins
- Aggregations
- Group By
- Subqueries
- KPI calculations

### 5. Power BI Dashboard
Built an interactive dashboard including:
- Revenue Analysis
- Sales by Category
- Customer Segmentation
- KPI Metrics

## Key Insights
- Clothing category generated the highest sales
- Young Adult customers contributed the highest revenue
- Most customers were non-subscribers
- Average customer rating remained strong across categories
